# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行垃圾分类检测
- 提供4类垃圾识别：危险废物、厨房垃圾、不可回收垃圾、可回收垃圾
- 支持选择模型和自定义置信度、IOU阈值
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 提供10000张标注数据集
- 包含训练好的模型及训练结果
- 配套完整的项目运行教程

## 适用场景
- 垃圾分类管理系统的开发
- 环保教育展示平台
- 智能垃圾分类设备开发
- 垃圾分类相关研究项目

## 技术特点
- 基于YOLOv8目标检测算法
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含完整UI界面源文件和图标
- 提供完整的程序文件和项目文档
